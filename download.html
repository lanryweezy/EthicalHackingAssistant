Installation Instructions

Windows Installation

  1. Download the WhiteHat installer (.exe or .msi)
  2. Run the installer as Administrator
  3. Follow the installation wizard
  4. Launch WhiteHat from the Start menu
# Alternative: Install via winget
winget install WhiteHat.Terminal

macOS Installation

  1. Download the WhiteHat .dmg file
  2. Open the .dmg file
  3. Drag WhiteHat to your Applications folder
  4. Launch WhiteHat from Applications
# Alternative: Install via Homebrew
brew install --cask whitehat

Linux Installation

  1. Download the appropriate package (.deb or .rpm)
  2. Install using your package manager
  3. Launch WhiteHat from the terminal or applications menu
# Ubuntu/Debian
sudo dpkg -i whitehat_1.0.0_amd64.deb
sudo apt-get install -f

# CentOS/RHEL
sudo rpm -ivh whitehat-1.0.0-1.x86_64.rpm
